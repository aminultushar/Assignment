HTTPS: Hypertext transfer protocol secure. It is a protocol for securing

Answer 2:
RFC (stands for Request For Comments) is a document that describes the
standards, protocols, and technologies of the Internet and TCP/IP. Requests for
Comments are produced in a non-reflowable document format, but work has begun
to change the format to a reflowable one so that documents can be viewed in
devices with restricted size.
